INSTALL (players)
  Double-click PLAY.bat at repo root → /reload in game

AUTO ON FIRST LOGIN
  Questie (±4 levels, proximity sort, auto accept/turn-in, map ! icons + objectives)
  P1QuestNav (numbered minimap pins + dotted line + TomTom arrow to #1)
  P1QuestPath logic (xp + gear ranked NEXT in guide — /p1path still toggles data)
  P1 Druid Guide — NEXT / MATS / GATHER / BIS overlay (/p1 or /p1guide)
  P1AutoQuest (accept/turn-in assist when Auto Q on)

TROUBLESHOOTING
  No guide?                   /p1guide — ensure ON; drag title bar to reposition
  No NEXT quests?               /p1path refresh — needs Questie loaded
  No map icons?                 /p1questie  then open world map
  No quest pins?                /p1nav debug — zone, spawn, placement mode
  Old range/damage addons?      PLAY.bat moves them to _disabled/

COMMANDS
  /p1guide or /p1     Toggle druid guide overlay
  /p1guide reset      Reset position and size
  /p1guide scale 0.8  Scale overlay
  /p1settings         Feature toggles (Auto Q, Nav, Path, Guide)
  /p1nav              Toggle minimap pins + TomTom
  /p1nav debug        Zone id, spawn coords, pin placement for #1
  /p1path             Toggle path scoring (feeds guide NEXT section)
  /p1auto             Toggle auto accept/turn-in
