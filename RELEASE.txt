  Double-click PLAY.bat at repo root → /reload in game

AUTO ON FIRST LOGIN
  Questie (±4 levels, proximity sort, auto accept/turn-in, map ! icons)
  P1QuestNav (NEXT line + exp/min pins + dotted line + TomTom arrow to #1)
  P1QuestPath (optimal route panel — xp + gear, /p1path)
  P1RangeDisplay (target distance number — default ON, /p1range debug)
  P1DamageText (floating damage numbers — default ON, /p1dmg)
  P1AutoQuest (accept/turn-in assist when Auto Q on)
  P1 Adventure Guide — crafting mats + AH tips (/p1guide)

TROUBLESHOOTING
  No distance number?          /p1range debug — check target + interact tiers
  No damage numbers?           /p1dmg — toggle ON; hit a mob to test
  No NEXT line?                /p1nav — ensure ON; needs Questie quest data
  No map icons?                /p1questie  then open world map
  No quest pins?               /p1nav debug — check ranked list + scores
  No optimal path?             /p1path refresh — needs Questie loaded
  Stuck TomTom arrow?          /p1fix
  Hide range display:          /p1range
  Hide damage text:             /p1dmg
  Hide mats panel:             /p1guide
  Hide path panel:              /p1path
  Toggle all features:          /p1settings  ·  /p1settings all off
  Toggle nav only:              /p1nav
  Auto quest debug:             /p1quest
  Minimal addon list:           /p1minimal
