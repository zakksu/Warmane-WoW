INSTALL (players)
  Double-click PLAY.bat at repo root → /reload in game

AUTO ON FIRST LOGIN
  Questie (±4 levels, proximity sort, auto accept/turn-in, map ! icons + objectives)
  P1QuestNav (numbered minimap pins + dotted line + P1Waypoint arrow to #1)
  P1QuestGlow (gold pulse on quest mob nameplates — /p1glow)
  P1QuestPath logic (xp + gear ranked NEXT in guide — /p1path still toggles data)
  P1 Druid Guide — character-aware NEXT / TIPS / MATS / GATHER / BIS (/p1 or /p1guide)
  Auctionator (AH search from guide — open AH before clicking [AH] lines)
  P1AutoQuest (accept/turn-in assist when Auto Q on)

TROUBLESHOOTING
  No guide?                   /p1guide — ensure ON; left-drag title, right-drag header
  Overlay won't drag?         /p1guide reset then /reload — v1.6 fixes SetMovable
  Want arrow on quest?        Click [GO] in header or /p1guide go (P1Waypoint)
  AH search does nothing?     Open Auction House first, then click [AH] or /p1ah
  No AH prices in NEXT?       Run an Auctionator scan at AH, then /p1scan or /reload
  No glow on mobs?              /p1glow — ensure ON; need incomplete kill objective
  Glow stuck?                   /reload — clears on objective complete or mob death
  No NEXT quests?               /p1path refresh — needs Questie loaded
  No map icons?                 /p1questie  then open world map
  No quest pins?                /p1nav debug — zone, spawn, placement mode
  Old TomTom addon?             Disabled by PLAY.bat — P1Waypoint is built into P1QuestNav
  Old range/damage addons?      PLAY.bat moves them to _disabled/

COMMANDS
  /p1guide or /p1     Toggle druid guide overlay (character-aware per toon)
  /p1ah               Search top pending AH upgrade in Auctionator (open AH first)
  /p1ah scan          Cache visible AH browse listings to realm market DB (assist-only)
  /p1ah relist        Print relist suggestions from bag + cached prices (you post)
  /p1ah watch         Queue SHOP item searches when AH opens
  /p1scan             List pending AH upgrades + refresh buyout prices
  /p1tips             Toggle TIPS section (lvl bracket advice)
  /p1glow             Toggle quest mob nameplate glow
  /p1guide reset      Reset position and size
  /p1guide go         P1Waypoint arrow to top ranked quest (or [AH] if gear gap)
  /p1guide autogo on  Auto-set waypoint when top quest changes
  /p1guide scale 0.8  Scale overlay
  /p1settings         Feature toggles (Auto Q, Nav, Path, Guide, Glow, Tips)
  /p1nav              Toggle minimap pins + P1Waypoint arrow
  /p1nav debug        Zone id, spawn coords, pin placement for #1
  /p1path             Toggle path scoring (feeds guide NEXT section)
  /p1auto             Toggle auto accept/turn-in
