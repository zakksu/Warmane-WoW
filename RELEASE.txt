INSTALL (players)
  Double-click INSTALL.bat at repo root → pick pack → confirm Warmane path → log in → /reload
  Or: INSTALL_WARLOCK.bat / INSTALL_DRUID.bat / PhaseOne_*/INSTALL.bat

BUILD (maintainers)
  Run RELEASE.bat at repo root → creates .zip files

AUTO ON FIRST LOGIN
  Questie (±4 levels, proximity sort, auto accept/turn-in)

  Leatrix (auto repair, sell greys)
  P1 Warlock HUD / P1 Feral HUD (class DoT/rotation helper + Auto Q toggle)
  P1 Adventure Guide (/p1guide)
