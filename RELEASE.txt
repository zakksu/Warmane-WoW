Warlock pack:  PhaseOne_LevelingPack/

CHANGELOG (v1.1.4)
  - P1AutoQuest: auto TomTom arrow to nearest Questie objective (or turn-in NPC when done)
  - Gentle idle click-to-move toward arrow when Auto Q ON and idle 3s+ (stops in combat / on input)
  - /p1quest — debug status; /p1auto toggles Questie accept/turn-in + arrow + walk
  - Addon conflict fixes: Leatrix Automate quests/gossip forced Off (use Questie + P1AutoQuest)
  - P1AutoQuest bundled in both Warlock and Druid packs; /p1minimal char-select checklist

CHANGELOG (v1.1.3)
  - Questie auto-accept/complete: preset sets Auto Modifier to Disabled (fixes Control blocking automation)
  - /p1auto or /p1qauto — toggle auto quest accept/turn-in; "Auto Q" button on class HUD (green=on, grey=off)

CHANGELOG (v1.1.2)
  - Questie: auto-accept and auto-complete quests (presets re-applied on upgrade)

CHANGELOG (v1.1.1)
  - P1AdventureGuide — next best action, professions, mat counts, zone rares (/p1guide)
  - /p1fix — clears stuck TomTom arrow and pauses stuck WeakAuras glows/icons
  - WeakAuras optional — P1 class HUDs work without importing auras
  - Docs: Adventure Guide mockups, stuck-aura troubleshooting, class tip links

INSTALL (players)
  Double-click INSTALL.bat at repo root → pick pack → confirm Warmane path → log in → /reload
  Or: INSTALL_WARLOCK.bat / INSTALL_DRUID.bat / PhaseOne_*/INSTALL.bat

BUILD (maintainers)
  Run RELEASE.bat at repo root → creates .zip files

AUTO ON FIRST LOGIN
  Questie (±4 levels, proximity sort, auto accept/turn-in)
  P1AutoQuest (TomTom arrow + idle walk when Auto Q on)
  Leatrix (auto repair, sell greys)
  P1 Warlock HUD / P1 Feral HUD (class DoT/rotation helper + Auto Q toggle)
  P1 Adventure Guide (/p1guide)

TROUBLESHOOTING
  Stuck WA icon or glow?  /p1fix
